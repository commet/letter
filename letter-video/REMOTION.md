# Remotion 기능 정리

식전영상 프로젝트에서 활용 가능한 Remotion 기능 전체 리스트.
새 기능을 붙일 때는 이 문서에서 골라서 요청하면 됩니다.

---

## Part 1 — Remotion Studio (프리뷰 UI)

브라우저 기반 개발·미리보기 툴. 현재 `http://localhost:3000` 에서 실행 중.

### 핵심 (가장 자주 쓰게 될 것)

#### 🎬 Timeline & Preview
- 타임라인에서 프레임 단위로 스크럽 (드래그로 위치 이동)
- 스페이스바 재생/정지, ← → 키로 한 프레임씩 이동
- JKL 키로 역재생·정지·재생 (편집 프로그램처럼)
- 0–9 키로 재생 속도 조절
- `I` / `O` 키로 구간 선택 (In/Out)
- 루프 재생

#### 🔄 핫 리로드
- `manifest.ts`에서 숫자 하나만 바꿔도 타임라인 즉시 갱신
- 코드 저장 → 브라우저가 0.5초 안에 새 결과 반영
- 현재 재생 위치는 유지되므로 "이 순간이 어떻게 바뀌나" 비교하기 편함

#### 📋 Compositions 패널 (좌측)
- 여러 영상을 한 프로젝트에 등록 가능 (예: `Main`, `Act1Only`, `Act6Fast`)
- 각각 해상도·FPS·길이 다르게 가능
- 개별 Sequence 클릭해서 그 구간만 프리뷰도 가능

#### ⏱️ Sequence 리스트
- 타임라인 하단에 66장 사진이 각각 하나의 Sequence로 보임
- 클릭하면 해당 사진의 시작 프레임으로 점프
- `"1. Act 1 — 어항 앞 어린 신랑"` 같은 이름 표시됨

### 디버그·편집 기능

#### ⚙️ Props Editor
- 컴포넌트에 props를 지정해서 즉석에서 값 바꿔가며 테스트
- 예: 줌 배율 슬라이더, 텍스트 내용 입력 필드

#### 🎨 Inspector / Measure tool
- 사진 위의 픽셀 좌표를 측정 → 얼굴 focal point 좌표 뽑을 때 유용
- 캔버스에 마우스 올리면 좌표 표시

#### 📸 Still Frame Render
- 현재 보고 있는 프레임 한 장만 PNG로 내보내기 (썸네일)

#### 🎞️ Render 버튼
- 우상단 "Render" 버튼으로 MP4/WebM 출력
- 해상도, 비트레이트, 포맷, 품질 지정 가능
- 진행률 바 표시

### 유용한 부가 기능

- **Performance panel**: 느린 프레임 찾기 (4K + 복잡 애니메이션일 때)
- **Safe zones**: TV/프로젝터 safe area 가이드 표시 — 식장 프로젝터용
- **Keyboard shortcuts**:
  - 재생: `Space`
  - 처음/끝: `Home` / `End`
  - 10프레임 점프: `Shift + ←/→`
  - 타임라인 줌: `Ctrl + +/-`
- **Share 렌더 프리셋**: 720p/1080p/4K 여러 프리셋 저장

## Part 2 — 고급 기능 (영상에 추가 가능한 것들)

### ✍️ 텍스트·타이포그래피

#### 기본 텍스트 오버레이
- 위치·크기·색·그림자 자유 (React CSS 그대로)
- 한글 폰트 로딩: `@remotion/google-fonts` → Noto Sans KR, Nanum Myeongjo, Gaegu 등
- 커스텀 폰트 파일 (.ttf/.otf) 업로드 가능

#### 애니메이션 텍스트
- 타이핑 효과 (한 글자씩 나타남)
- 단어별 페이드인 (연설문 같은 느낌)
- 마스크 리빌 (아래→위로 슬라이드 등장)
- 숫자 카운트업 (예: `2012 ↑ 2026`)
- Spring physics 기반 튀어오르는 텍스트

#### 자막·캡션
- `@remotion/captions` 플러그인: SRT 파일 자동 파싱
- 현재 프레임에 맞는 자막 자동 표시
- 각 Act 시작 때 "2012, 대학생 시절" 같은 챕터 타이틀

#### 지금 바로 붙일 수 있는 것
- Act 시작마다 제목 카드 (예: "1999 — 같은 공동체에서")
- 사진 코너에 연도 뱃지
- 엔딩 타이틀: "2026.XX.XX / 신랑 ○○ · 신부 ○○ / 와주셔서 감사합니다"

### 🖼️ 프레임·필터·질감

#### 포토 프레임
- 폴라로이드 스타일 (하단 캡션 공간 포함)
- 빈티지 필름 프레임 (스프로켓 구멍)
- 고전 명화 액자 (황금 테두리)
- 커스텀 마스크 (원형, 하트, SVG 자유)

#### 필터·효과
- 세피아/흑백/필름 그레인
- 비네트 (가장자리 어둡게)
- 라이트 릭 오버레이 (8mm 필름 느낌)
- 먼지·스크래치 (아날로그 질감)
- 블러 배경 (현재 적용 중)

#### 파티클 효과
- 반짝이 (엔딩 축복 느낌)
- 눈 내림 (겨울 사진에)
- 꽃잎 떨어짐 (벚꽃 사진과 어울림)
- `@remotion/shapes` 또는 커스텀 Canvas로 구현

### 🔄 고급 전환 효과

`@remotion/transitions` 패키지로 제공:

- **Fade** (기본, 현재 사용 중)
- **Slide** (좌→우, 상→하 등 4방향)
- **Wipe** (8방향 + 대각선)
- **Clock Wipe** (시계 방향 회전)
- **Iris** (원형 열림/닫힘) ★ 결혼식 영상에 잘 어울림
- **Flip** (3D 카드 뒤집기)
- **Custom GLSL shader** (파도, 글리치 등 완전 자유)

"Act 간 전환은 Iris, Act 내부는 크로스페이드" 식으로 섞을 수 있음.

### 🎞️ 레이아웃·합성

#### Split screen / 병렬
- 화면 2분할로 신랑 어린시절 ↔ 신부 어린시절 동시 표시
- Act 1 "각자의 처음" 섹션에 넣으면 감정 임팩트 강함

#### Picture-in-Picture
- 큰 사진 안에 작은 사진 띄우기
- 예: 단체사진 크게 + 본인 얼굴 크롭을 구석에 작게

#### 그리드 몽타주
- 여러 장을 타일처럼 배치
- Act 2 교회 유년기 여러 장을 한 프레임에 모자이크 → 한 장씩 확대

#### 얼굴 자동 크롭
- OpenCV.js / MediaPipe 연동해서 얼굴 자동 감지
- Ken Burns를 얼굴 중심으로 자동 지정
- 단체사진에서 본인 강조할 때 유용 (수동 좌표 지정보다 빠름)

### 🎵 오디오 (BGM 확정되면)

#### 비트 싱크
- `@remotion/media-utils`로 오디오 파형 분석
- 특정 BPM에 맞춰 컷 타이밍 자동 조정
- Act 6 크리스마스 공연을 진짜 비트에 맞춰 떨어지게

#### 오디오 페이드·덕킹
- 인트로 페이드인, 엔딩 페이드아웃
- 내레이션 들어올 때 BGM 자동으로 작아지게

#### 볼륨 자동 조절
- 프레임마다 volume 커브 지정 가능

### 🎨 모션 그래픽

#### Lottie 애니메이션
- After Effects에서 만든 JSON 애니메이션 삽입
- LottieFiles에 결혼 관련 무료 애니메이션 많음 (반지, 비둘기, 꽃 등)

#### SVG 애니메이션
- 손글씨처럼 그려지는 글자 (stroke-dasharray 애니메이션)
- 심플한 일러스트레이션 등장

#### 3D (Three.js)
- 사진을 3D 공간에 배치하고 카메라 이동
- 종이 위에 놓인 사진이 넘어가는 효과
- 오버킬이지만 원하시면 가능

### 🗂️ 프로젝트 구조

#### 여러 Composition
- 한 프로젝트에서 여러 영상 등록
- 예: `Main` (풀 영상), `TrailerShort` (30초 티저), `ActSix` (공연 부분만)
- 각각 독립적으로 렌더 가능

#### 파라미터 기반 렌더
- CLI로 변수 주입해서 다른 버전 뽑기
- 예: `--props='{"language":"en"}'` → 영어 자막 버전

#### 프레임 범위 렌더
- Act 6만 뽑아서 확인 → 빠른 피드백 루프

## Part 3 — 이 프로젝트 우선순위

임팩트 대비 구현 시간 기준:

1. **Act 타이틀 카드** — 각 Act 시작에 "1999 — 같은 공동체에서" 류 (★ 빠르게 작품 느낌)
2. **엔딩 타이틀** — 신랑·신부 이름, 날짜, 감사 메시지
3. **Iris 전환** — Act 1 → Act 2 넘어갈 때 공동체로 "열리는" 느낌
4. **Split screen** — Act 1 신랑/신부 교차편집
5. **Lottie 반지·비둘기** — 엔딩 장식
6. **폴라로이드 프레임** — Act 2 교회 유년기 (필름 느낌)
7. **얼굴 자동 크롭** — 단체사진에서 본인 강조
8. **비트 싱크** — BGM 확정 후

> 1 + 2 + 4 조합이 가장 가성비 좋음. 이 3개만 붙여도 영상이 확연히 "작품"이 됨.

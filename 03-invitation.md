# 03. 청첩장 (Invitation)

## 현재 상태: 진행중 (방향 수립 완료, 구체적 업체 선정 전)

## 기본 방향

- **종이 청첩장**: 100~150부 (어르신, 직장 상사, 윗분들 위주)
- **모바일 청첩장**: 전체 하객 대상 (약 250명), 카카오톡 배포
- 종이에 QR 코드 삽입하여 모바일 청첩장으로 연결

## 종이 청첩장

### 가격대별 업체

| 등급 | 업체 예시 | 장당 가격 | 100장 기준 | 특징 |
|------|----------|----------|-----------|------|
| 저가형 | 카드마을, 보자기카드 | 600~1,200원 | 6~12만원 | 온라인 주문, 템플릿 |
| 중간형 | 바른손카드, 디얼디어, 잇츠카드 | 1,000~2,500원 | 10~25만원 | 모바일 무료 제공 포함하는 곳 많음 |
| 프리미엄 | 커스텀 디자인 | 3,000원+ | 30만원+ | 디자인비 별도 |

### 필수 포함 내용
- 신랑·신부 이름
- 혼주 성함
- 예식 일시 (2026년 5월, 정확한 날짜)
- 예식 장소: 서울대학교 교수회관
- 찾아오시는 길 (약도)
- 모바일 청첩장 QR 코드

### 권장 타임라인
- D-90: 업체 선정 + 디자인·문구 확정
- D-75: 시안 확인 및 수정
- D-60: 인쇄 완료
- D-45~30: 종이 청첩장 배부 시작

## 모바일 청첩장

### 방식 비교

| 방식 | 장점 | 단점 | 비용 |
|------|------|------|------|
| 유료 플랫폼 (데어무드 등) | 빠르고 예쁨, RSVP 기능 | 커스텀 한계 | ~19,800원~ |
| 셀프 코딩 (Vite + Vercel) | 완전 커스텀, 무료 | 개발 시간 필요 | 무료 |
| 종이 업체 모바일 번들 | 종이와 함께 제작 | 디자인 제한 | 종이 가격에 포함 |

### 모바일 청첩장 필수 기능
- 메인 이미지 + 날짜
- 예식 장소 정보 + 지도 (카카오맵/네이버맵 연동)
- 축의금 계좌번호
- 참석 여부 회신 (RSVP)
- 갤러리 사진
- 카카오톡 공유 버튼

### 카카오톡 배포 시 주의사항
- **카카오 JS SDK** 연동하면 공유 시 썸네일이 깔끔하게 노출됨
- 단순 URL 붙여넣기보다 공유 API 사용 권장
- **모르는 URL을 피싱으로 오해**하는 어르신이 많으므로 카카오 공유하기 기능 필수
- 발송 후 내용 수정 시 **카카오 캐싱** 때문에 이전 버전이 노출될 수 있음 → 최종 확정 후 발송
- OG 메타 태그 설정 꼼꼼히
